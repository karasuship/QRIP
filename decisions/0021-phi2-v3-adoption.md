# 0021: phi2 v3 採用決定

**日付**: 2026-06-22  
**状態**: 採用  
**前決定**: 0019（phi2 v2 cap6 採用）

---

## 決定事項

phi2 v2 に以下の2フィルタを追加した **phi2 v3** を採用する。

```
phi2 v3 = phi2 v2 AND age_ath NOT IN [91, 252] AND CRS >= 2
```

**phi2 v2 から変わること**: age 91〜252 日のトリガーをスキップ + CRS <= 1 のトリガーをスキップ。

---

## 根拠

### phi2 v2 の弱点の解明（Round 19〜30）

phi2 v2 は TRAIN Z=-0.16（有意でない）だった。30ラウンドの研究で根本原因が判明した:

1. **age 91-252（GFC L字ゾーン）**: TRAIN Z=-6.65。このゾーンの phi2 は63日ホライズンでほぼ全て負ける。GFC 長期低迷中（最大252日間）に資金が入り続ける設計上の問題。
   - 分割買いでも改善しない（R26）
   - 答えはスキップ（R26）

2. **CRS <= 1（恐怖回復兆候局面）**: TRAIN Z=-2.85。恐怖指標が既に改善し始めた後の phi2 は品質が低い（R28）。

この2集団を除外すると:

| 指標 | phi2 v2 | phi2 v3 | 改善 |
|---|---|---|---|
| TRAIN Z | -0.16 | **+4.54** | +4.70 |
| TEST Z | +8.04 | **+8.28** | +0.24 |
| TEST DCA差 | +5.83% | **+7.17%** | +1.34% |
| TRAIN 勝率 | 55% | **70%** | +15% |
| TRAIN n | 136 | **73** | -63（フィルタ後） |

### Bonferroni 基準との整合

累計~290テスト → 閾値 Z~3.80。phi2 v3 TRAIN Z=+4.54 は超えている。

補注: この仮説は R26/R28 の副産物の組み合わせであり、完全な独立テストとは言えない側面がある。
ただし:
- 理論的根拠が明確（GFC L字の構造的問題、恐怖継続中の高品質局面への絞り込み）
- GFC除外 Jackknife でも TRAIN Z=+5.14（強い）
- 4体制 v3 での確認で TRAIN Z=+3.97（独立確認）

### CRS 成分の定義（R28）

| 成分 | 条件 | TRAIN Z（=1） |
|---|---|---|
| C1 | VIX > 30 | +2.02 |
| C2 | HYG 3日リターン <= 0 | +1.31 |
| C3 | DXY 5日上昇 | +0.05 |
| C4 | age_ath <= 90 | **+5.00** |
| C5 | HYG 60日高値-8%以下 | -1.58 |
| C6 | RSP 5日相対リターン <= 0 | +0.21 |

CRS = C1+C2+C3+C4+C5+C6（0〜6点）。>= 2 が発動条件。

## 4体制への影響

4体制 v3 = phi2 v3 + RSI<25 + HYG-8% QE後 + B4 の OR

| 指標 | 4体制 v2 | 4体制 v3 |
|---|---|---|
| TRAIN Z | +1.59 | **+3.97** |
| TEST Z | +14.81 | **+14.79** |
| TEST DCA差 | +6.39% | **+6.70%** |
| GFC除外 TRAIN Z | +2.81 | **+3.91** |

4体制 v3 は TEST を犠牲にせず TRAIN を大幅改善。Bonferroni 閾値近辺（Z~3.82 vs 実測 Z=+3.97）。

## 実装上の留意点

phi2 v3 をシグナル判定するには、当日以下のデータが必要:
- **VIX**: C1 判定
- **HYG**: C2（3日リターン）、C5（60日高値比）、age_ath
- **DXY**: C3（5日変化）
- **RSP + SPY**: C6（5日相対リターン）—2003年以降のみ
- **days_since_ATH**: age_ath（ATH からの日数）

`/signal` ページ実装では Yahoo Finance から当日取得。
