# 0024: データ観測システムの設計思想（2026-06-22）

## 発端となった認識

Round 37-43 の検証で「殺した」シグナル（TNX / IRX / CAPE<20 / LQD 等）は
「間違っていた」のではなく「体制が変わった局面では機能しなかった」だけ。

**核心**: 市場には「体制（レジーム）」の切れ目がある。
シグナルの有効性はその体制に条件付けされている。
体制を判定できれば、殺したシグナルを正しい文脈で復活させられる。

さらに: **今観測を始めなければ、永遠に入手できないデータが存在する。**
これは将来の分析資産として今から転がし始めるべき「無形資産」。

---

## データの価値序列

### 1次データ（誰でも持てる・差別化ゼロ）
価格・出来高・経済統計の生値。必要条件だが競争優位にならない。

### 2次データ（計算で生まれる・部分的に差別化）
CRS・VIX ターム構造・信用スプレッド階層・セクター相対強度など。
計算式があれば再現できるが、VIX9D（2011年以降のみ）など一部は歴史的に取れない。

### 3次データ（行動・反応・文脈の記録 → ここが本命）

市場が「情報をどう処理しているか」を記録したもの。価格ではなく**解釈の記録**。

```
例1: 発表値 vs 市場反応の驚き係数
  NFP +28万人（予想比-2万）→ 市場-1.2%
  「類似発表での平均反応は-0.3%」→ 今日は悲観が過剰 = 逆張り示唆

例2: FOMC声明 vs 市場の初期反応 vs 2週間後の乖離
  タカ派声明 → 翌日-2% → 2週後+5%  （市場は過剰反応した）
  タカ派声明 → 翌日-2% → 2週後-8%  （市場は正しく反応した）
  → この差の時系列が「Fed 信頼度指数」になる

例3: VIX と リターンの解離
  SP500 -0.5% なのに VIX +15% → 恐怖がプライスを先取り（先行指標）
  SP500 -2.0% なのに VIX +5%  → 恐怖が鈍い → まだ売り切れていない

例4: セクターローテーション速度
  危機時にどのセクターが最初に売られ、最初に回復するか
  → 危機の「タイプ分類」器になる（流動性危機 vs インフレ危機 vs 信用危機）
```

**なぜ3次データが「無形資産」か:**
- 過去に遡って再現できない（行動・反応パターンは蓄積しないと生まれない）
- 市場参加者の心理変化を定量化できる数少ない手段
- 将来の「体制条件付きシグナル」の訓練データになる

## 構築フェーズ計画

### Phase 1（即着手・コスト ¥0）
日次バッチ: 40ティッカー取得 → Supabase に1行 append  
含む: 価格 + vol構造 + スプレッド + セクター + CRS 日次計算値

優先ティッカー:
- vol構造: ^VIX9D / ^VIX / ^VIX3M（近期/遠期の恐怖比）
- テールリスク: ^SKEW / ^MOVE（株以外の市場が何を警戒しているか）
- 信用階層: HYG / LQD / JNK / TLT
- セクター×11: XLK/XLF/XLE/XLV/XLI/XLY/XLP/XLU/XLRE/XLB/XLC
- 市場幅: RSP / IWM / MDY
- 国際: EFA / EEM / FXI
- 金利形状: IRX / FVX / TNX / TYX
- 実物: GLD / USO / DBA
- 為替: DXY / USDJPY

### Phase 2（来月・コスト ¥0〜500/月）
イベント記録: FOMC・CPI・NFP ごとに
  - 発表値と予想の乖離（驚き係数）
  - 1時間後・1日後・2週間後の市場反応
  - 発表時の体制ラベル
→ 蓄積すると「このタイプの驚きが体制Xで発生したときの期待リターン」が出る

### Phase 3（3ヶ月後・Claude Haiku API ~¥150/月）
毎日22時バッチ:
  - その日の主要ニュース5本 → Claude でスコアリング
  - 出力: {sentiment, crisis_relevance, fed_tone, topics} JSON
月次:
  - FOMC声明全文 → Hawk/Dove スコア + 前回比変化

### Phase 4（半年後〜・計算のみ）
蓄積データに「体制ラベル」を付与し
殺されたシグナルを体制別に再検証する
→ EV = シグナル固有EV × 体制一致確率 × ポジション上限

## 将来の活用イメージ（EV+ 定量化）

```
博打も期待値+で死なないレベルの分散なら乗るべき

現状: phi2 v3 の TEST Z=+8.65 → EV が定量化されている
課題: 体制が変わったときの EV が不明

目標:
  EV(phi2 | 体制=QE体制)     = +14.5%（TEST 実績）
  EV(phi2 | 体制=引き締め)   = +X%（未検証・蓄積待ち）
  EV(TNX急騰 | 体制=正常金利) = +Y%（TRAIN 実績あり・体制ラベル付与で復活）

→ ポジションサイズ = Kelly比率 × 体制一致確率
```
