# 決定 0013: 急落買い増しルール（D2、暫定採用）

- **日付**: 2026-06-20
- **根拠**: engine/archive/0060-0061
- **状態**: 採択・暫定

## 決定内容

通常の月次DCAに加え、「-2%急落日に0.5単位を追加投入する」ルールを暫定採用する。

### 具体的なルール

```
毎月: 月初第1営業日に 1 単位（基本積立）
条件: 当日の騰落率 <= -2.0% かつ その月まだ追加未実施
行動: その日に 0.5 単位を追加投入
上限: 月1回（翌月リセット）
```

### 検証済み効果（OOS TEST 2011-2026）

| 指標 | 値 |
|---|---|
| DCA対比ROI改善 | +0.25% |
| ボーナス投入分単独CAGR | +7.60%（vs DCA +7.47%） |
| ボーナス1円あたりの超過ROI | +1.75% |
| 月あたりのボーナス発生頻度 | 0.33回/月（約3ヶ月に1回） |

### 根拠

- D2のボーナス投入はDCA通常投入より1.75%高いROIを達成（OOS）
- FULL(+1.46%)・TRAIN(+0.41%)・TEST(+0.25%)すべてプラスで一貫性あり
- -2%急落は「ノイズではなく意味のある局所安値」（-1%は逆効果だった）
- 待機コスト（drift）が発生しない構造（落ちてから買うため）

## 制約・注意事項

- 月1回上限: 大クラッシュ（連続-2%が複数日）の際に追加機会を逃す
- 必要資本: 月次積立の33%程度の予備資金が必要
- -1%トリガーは使わない（TEST -0.45%と劣化）
- D2_Wは無効（W条件月 = 低ボラ = -2%急落が来ない = 反相関）
- D3(-3%)は1ボーナス当たりの効果大きいがn=20と薄い（将来サンプルに期待）

## 2026-06-21 追加検証（0062）

### 月次上限の重要性が確定

| 戦略 | TRAIN | TEST |
|---|---|---|
| D2 cap=1 | +0.41% | +0.25% |
| D2 uncapped | **-7.56%** | +2.44% |

上限なしは2008年(41回)・2002年(29回)に無制限の落ちるナイフ掴みとなり、TRAINで壊滅的に劣化。  
**月次cap=1はリスク制御として必須。** 上限撤廃は採用しない。

### volフィルタでD2を改善

- D2単独: TRAIN +0.41% / TEST +0.25%
- **D2 × vol>p50 (H6): TRAIN +0.45% / TEST +0.44%** → より安定
- D2 × vol>p75 (H6b): TRAIN +0.38% / TEST +0.64% → TEST高め

H6（vol>p50フィルタ）はD2の改善版。「ボラが低い穏やかな日のノイズ的-2%」を除外できる。

### 週次DCAとの組合せ

- 週次D2 cap=1: TRAIN +0.57% / TEST +0.44% → 月次より若干安定

### ボーナス単独ROI（TEST）最強トリガー

H2b（ATH-10%）: bonus_ROI = 1.1684（+16.84%超過）が最大  
ただし TRAIN -0.29%（2001-2009の弱気相場で逆風）

## 採用ルール（更新版）

```
基本: 毎月 月初第1営業日に 1 単位
追加: 当日-2%以下 AND vol20 > 中央値 かつ月内未使用
行動: 0.5 単位追加（月1回上限・翌月リセット）
```

※ volフィルタ中央値 ≒ 0.135（FULL中央値、毎年更新不要）
