# 0033: NTT 配当利回りシグナル 実装

**日付**: 2026-06-26  
**状態**: 実装中  
**提起者**: 竹田凜士郎

---

## 背景・根拠

decisions/0032（権利落ち後過剰落ち研究）の過程で、NTT（9432）の分析を通じて以下が判明した：

- **過剰落ち仮説は棄却**（Round 45）
- しかし**配当利回りベースの割安・割高判断**に統計的有意性を確認
  - 利回り ≥ 3.5% で買い → ≤ 3.0% で売り: 平均 +7.7%, Z=1.88, n=20（26年間）
  - 利回り ≥ 3.5% 時に 12ヶ月保有: 平均 +24.6%
  - 52週レンジ下位20%でのエントリーが最も確度が高い
- 価格が目標に達しなくても配当（年 3.5-5%）で損失をカバーできる構造

---

## シグナル定義

### 買いシグナル（NTT_BUY）
```
条件: NTT 配当利回り ≥ 3.5%
      AND 52週レンジ位置 ≤ 20%（下位20%）
```

### 売りシグナル（NTT_SELL）
```
条件: NTT 配当利回り ≤ 3.0%
      OR 52週レンジ位置 ≥ 80%（上位80%）
```

### パラメータ根拠
| パラメータ | 値 | 根拠 |
|---|---|---|
| 買い利回り | 3.5% | 過去26年の80パーセンタイル相当（割安ゾーン）|
| 売り利回り | 3.0% | 過去26年の55パーセンタイル（やや割高ゾーン）|
| 52週下位 | 20% | 安値圏を捉える保守的フィルター |
| 年間配当 | ¥5.30/株 | 直近実績（2026年時点） |

### 統計的根拠
- 利回り3.5%買い→3%売り: 平均+7.7%, 勝率60%, Z=1.88, n=20（26年）
- 利回り3.5%時に12ヶ月保有: 平均+24.6%, 勝率73%, Z高
- Z=1.88 は phi2（Z=+7.17）より弱い。「サブシグナル」として位置付け

## 実装方針

### データフロー
```
毎営業日7時 cron
  └── 9432.T 株価を Yahoo Finance から取得（1y 範囲）
       ├── 当日の生株価（raw close）
       ├── 52週高値・安値（250営業日）
       ├── 配当利回り = 年間配当(¥5.30) / 当日株価
       └── シグナル判定 → Supabase 保存 + 通知
```

### Supabase テーブル
`market_daily` に以下カラム追加（または既存 JSON フィールドに格納）:
- `ntt_price`: NTT 当日株価
- `ntt_div_yield`: 配当利回り（小数 e.g. 0.0368）
- `ntt_w52_lo`: 52週安値
- `ntt_w52_hi`: 52週高値
- `ntt_w52_pos`: 52週レンジ内位置（0.0-1.0）
- `ntt_signal`: "BUY" | "SELL" | "HOLD"

### 通知内容
```
【NTT 割安シグナル】
配当利回り 3.7%（26年90パーセンタイル）
52週レンジ 底圏 9%
根拠: 配当でリスク限定。上昇時は追加リターン。
```

## 表示（/signal ページ）

- phi2/CRS セクションとは独立したセクション「日本株（個別）」として表示
- 表示項目: 現在株価、配当利回り、52週レンジ位置、シグナル状態
- 次のステップ: 他銘柄（JT/KDDI等）も同一フレームで追加可能な設計にする

## QRIP サービスとの整合性

- decisions/0022（サービスコンセプト）: 「証拠と文脈の提示」に合致
  - 「今は割安かどうか」の客観的根拠を数字で示す
  - 買い・売りを「決定」せず「材料」として渡す
- phi2 とは独立したシグナル（SP500 暴落回復 vs 日本個別株サイクル）
- 将来的に他の高配当日本株（JT: 2914、KDDI: 9433等）へ横展開可能な設計

## 注意事項

- 年間配当 ¥5.30 はハードコードで始める（配当変更時は手動更新）
- Yahoo Finance の raw close（調整なし）を使う（配当込みでない純粋な市場価格）
- NTT は 2023年に 25:1 株式分割済み。現在の株価スケールは ¥130-200 帯
- 配当政策変更・減配リスクは注記として表示
