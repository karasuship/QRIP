## 発見①: RSI<25 は「短期アルファなし」だが「長期価値あり」

### 矛盾に見える数字の正確な解釈

| 指標 | 数値 | 意味 |
|---|---|---|
| RSI<25 TRAIN Z | +0.06 | 63日後のリターンはランダムと統計的に区別できない |
| RSI<25 追加投資額（30年） | 177万 | OR - phi2only の差 |
| RSI<25 が生んだ追加利益 | +843万 | 6,780万 - 5,937万 |

**矛盾の解消:**  
Z=+0.06 は「RSI<25 が発動した63日後のリターンが、ランダムな63日後より優れるかどうか」を測る。  
これはほぼゼロ → RSI<25 に「短期反発の予測力」はない。

だが「暴落時に安値で買った株を30年保有したらどうなるか」は別の問い。  
RSI<25 = 大幅な下落局面 → 平均より低い価格で購入 → 30年後に大きく育つ。  
**この価値は「長期buy-the-dip効果」であり、Z スコアが測っていないものだ。**

### 実装への示唆

RSI<25 はシステムから外さない。ただしサービス上の表現を変える：

| 変更前（誤） | 変更後（正） |
|---|---|
| RSI<25 ＝ 反発シグナル | RSI<25 ＝ 深い恐怖局面の記録（反発予測ではない） |
| phi2 と同格のシグナル | 「phi2 も同時発動なら最高品質。単独発動は低信頼度」 |
| シグナル = 買え | 「DCA 積立の追加タイミングとして統計的に合理的」 |

### UI 品質ティア（実装案）

```
Tier A: phi2 v3 + CRS=5 + RSI<25（三重一致）
  → 「超高品質: 過去30年8回。63日平均 +20.5%」

Tier B: phi2 v3（RSI>=25）
  → 「高品質: 年平均3回。63日平均 +13.6%」

Tier C: phi2 v3（RSI>=25、CRS=2-3）
  → 「標準シグナル: 年平均2-3回。63日平均 +8-10%」

Tier D: RSI<25 のみ（phi2 条件なし）
  → 「低信頼度シグナル: 短期反発予測力なし。長期保有前提なら有効」
```

## 発見③: EFA（先進国）並列 phi2 → +360万 / 30年

### 数値

| | SP500 phi2 のみ | SP500 + EFA 並列 |
|---|---|---|
| 追加投資 | 108万 | 210万 |
| 30年後 | 5,937万 | 6,297万 |
| 純DCA比 | +702万 | +1,062万 |
| 差 | — | +360万（EFA 分） |

EFA 専用シグナル: 88件（FULL 30年）≒ SP500 の 93件と同水準  
EFA追加投資 102万 → 360万に成長（3.5倍）

### 重要: EFA シグナルは SP500 と異なる日付で発動する

SP500 phi2 と EFA phi2 は同じ CRS を使うが、ATH 乖離・ボラ・当日騰落は別計算。  
「SP500 が発動した日に EFA も買う」とは異なる（EFA 独自のシグナルが存在する）。

### 実装段階

**Phase 2（中期）で実装可能:**
1. EFA の株価データは既取得（`engine/data/EFA_stock.csv`）
2. phi2 ロジックは SP500 と同じコードを EFA に適用するだけ
3. `/signal` ページに「グローバル（EFA）」タブを追加

**表示案:**
```
本日のシグナル状況
  SP500:  phi2 未発動  CRS=2
  EFA:    phi2 発動中  ATH-16.2%  CRS=2  [Tier B]
  
EFA シグナル: 先進国（欧州・豪州・日本）の phi2 シグナル
統計的根拠: 30年 TEST Z=+8.08, 平均+15.6%（SP500 とほぼ同等）
```

## 決定事項

### 採用

1. **RSI<25 はシステムに残す**。ただし Tier D（低信頼度）として表示
2. **UI 品質ティア（A/B/C/D）を実装**。シグナル発動時の文脈を豊かにする
3. **ATH 深度インジケーター**: -15%〜-20% ゾーンを「最高品質帯」として表示
4. **EFA 並列シグナルを Phase 2 で実装**（データ取得済み・同ロジック）
