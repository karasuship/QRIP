# 書庫 0034: S&P内部構造（下位企業は負の吸収かブレーキか）＝等加重 vs 時価加重
（信頼設計＝decisions/0006。A「下位企業は上位の負を吸収か成長のブレーキか→指数を最適化で超えられるかに繋がる」）

## 結果（RSP等加重 vs SPY時価加重、2003-2026）

| era | RSP CAGR/vol/maxDD | SPY CAGR/vol/maxDD |
|---|---|---|
| FULL | +11.2% / 20% / -60% | +11.5% / 19% / -55% |
| 2003-2014 | **+11.9%** / 21% / -60% | +9.3% / 19% / -55% |
| 2015-2026 | +10.6% / 18% / -39% | **+13.8%** / 18% / -34% |

## 答え

1. **「負を吸収」＝No**：下位厚めのRSPはむしろ高vol・深いmaxDD（-60 vs -55）。下位企業はリスクを足す、吸収しない。
2. **「成長のブレーキ」＝時代次第**：2003-14は下位がプレミアム（等加重勝ち）、2015-26は下位がブレーキ（メガキャップ独走で時価加重勝ち）。

## 含意

- 「最適化で指数を超えられるか」＝原理的Yes、だが**どちらの時代かを当てる＝メガキャップ集中が続くかのレジーム賭け＝非定常問題**（散々"難しい"と示したもの）。
- 今(2015-26)は時価加重が勝ったが、継続は賭け。等加重/時価加重の選択自体が、集中度レジームへのベット。

## 地位
**S&P内部構造＝下位はリスク源・寄与は時代依存。** 指数最適化＝レジーム賭けに帰着＝EVの壁は同じ。Aの問いに直接回答。
