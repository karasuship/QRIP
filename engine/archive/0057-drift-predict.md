# 書庫 0057: ドリフト直接予測 — vol20が構造的連動を初めて切った

（検証軸③。0056まで全条件でP(drop)とdriftが連動してEV正転不可。逆から攻め「driftが低い局面」を予測する）

- **対象**: S&P500 × X=10% Y=63d
- **特徴量**: CAPE / vol20 / 200MA乖離 / term_spread / 複合スコア
- **OOS分割**: TRAIN=1996-2011 / TEST=2011-2026

## 重要発見: vol20が構造的連動を初めて切った

これまで全条件で「P↑ = drift↑」の連動が続いていた。

**vol<p25（低ボラ）では初めてP↓ × drift↓が同時に起きた:**

- UNC TEST: P=13%, drift=+4.6% → EV=-6.05%
- R(vol<p25) TEST: P=5%, drift=+3.4% → EV=-4.14%

P は 13%→5% に低下（待っても下落が来にくい）  
drift も 4.6%→3.4% に低下（機会損失が小さい）  
→ EV改善の方向が揃っている（初めて）

vol20が最大スロープ（かつFULL→TEST安定）: 高vol = 高drift（no drop時）

## 解釈

低vol期 = 「静穏相場」:
- 上値も限定（drift低い）
- 下落も来にくい（P低い）
- 結果: EV改善するが方向が「チャンスがない」に由来する

高vol期 = 「乱気流」:
- drift は逆説的に高い（下落がなければ急反発）
- P(drop) も高い
- 待機戦略は悪化する

→ **逆説: 「待機コストが最も低い時期 = 相場が最も動かない時期」**

## TRAIN vs TEST のギャップ

- R TRAIN: EV=-1.91%（ほぼブレイクイーブン！）
- R TEST: EV=-4.14%（劣化）
- 方向性は一致。完全なOOS一致ではないが、傾向はロバスト

## 結論

**vol<p25（低ボラ期）は全検証の中で最初に構造的連動を切った条件。**
ただしEV正転（>0）には届かず。残課題:

- low-vol × さらなる絞り込み（dist200>p75との組合せ等）
- S (CAPE>p75×vol<p25) TEST: EV=-3.49%, n=17 → 統計的に薄いが最良
- 実用化には「vol低い × 市場過熱」を組み合わせた戦術的タイミングとして位置づける可能性
