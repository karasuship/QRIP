# 研究結果の限界と正直な評価（2026-06-22）

> この文書は「数値が良く見える理由」を正直に残すためのもの。
> 採用シグナルの数値に過度な信頼を置かないための参照。

## 1. TEST 期間が「都合よく良い」

TRAIN: 1996〜2011（GFCを含む）
TEST: 2011〜2026（QE全盛期〜2022年利上げ回復）

TEST 期間（2011-2026）の特徴:
- 史上最長の強気相場（COVID急落を除き一方的上昇）
- FRB が株価を支持する「パウエルプット」文化
- COVID 暴落は史上最速の V 字回復（63日での判定が最も有利）
- 2022 年の利上げ相場も 2023-2024 年に完全回復

この環境では **「急落で買う」戦略はほぼ何でも有効に見える**。
TEST DCA差 +13.84% は、戦略の優秀さだけでなく TEST 期間の有利さも含んでいる。

GFC（2007-2009）が TRAIN 側に入っているのは単に機械的 mid 分割の結果だが、
「最も戦略が機能しない局面」が TRAIN に収まったことで TEST が clean になっている。
これは意図的ではないが、数値を膨らませる方向に働く。

## 2. B4 の TRAIN Z がマイナス

B4（phi2後7日後）: TRAIN Z=-0.67、TEST Z=+8.37

TRAIN でマイナスのシグナルを TEST Z だけで採用した。
「phi2 と同じ構造的問題だから許容」という説明は論理的だが、
TEST に現れた強さの一部は TEST 期間の有利さかもしれない。

## 3. HYG8 の後付けレジーム条件

HYG-8%は「全期間では TRAIN 逆転」→「QE 後（2009以降）に限定すれば有効」という経緯で採用された。

QE以降限定という条件は**データを見た後に設定した**。
これは典型的な「後付け最適化」に近い操作。
TEST Z=+9.42 の数値は、「QE後限定」という条件が QE 後の TEST 期間でバックフィットした可能性を排除できない。

「QE が構造変化だから」という説明は理論として合理的だが、
それ自体もデータを見た後の解釈。

## 4. 多重比較補正の限界

Bonferroni 補正（Z > 3.71）を適用したが:
- 180 テストのうち多くは相関（同じ SP500 データ、似た条件）
- Bonferroni は独立テストを仮定しており、相関テストには保守すぎる（閾値が低くなりすぎる）可能性もある
- 逆に、テスト設計自体（仮説生成→検証→次の仮説）は厳密な事前登録ではない

Z > 3.71 という閾値はあくまで「上限の参考値」であり、採用の絶対条件ではない。

## 正直な結論

**出た成果**:
phi2 v2 のコアロジック（ATH乖離 × 急落 × vol）が SP500 30年データで統計的に有意。
仕組みの説明（品質フィルタ × 恐怖の最大化）も理論的に合理性がある。

**出ていない成果**:
「+13.84% の DCA 超過パフォーマンスが将来も続く」という保証は出ていない。
TEST の数値は部分的に TEST 期間の有利な環境を反映している。

**適切な使い方**:
- phi2 がトリガーされたとき「少し多めに買う」という判断の根拠として使う
- 機械的に固定量を準備して毎回機械執行するほどの信頼性は、まだない
- GFC 型のシナリオ（日本型・L字型）を想定した場合の資金計画は別途必要
