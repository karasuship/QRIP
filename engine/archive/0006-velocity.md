# 書庫 0006: 下落の速さ（velocity）→ 1年後リターン
（信頼設計＝decisions/0006。A の「水準だけのどんぶり勘定」批判を受けた特徴量改良）

- **指紋**: `sp500_adjclose / trailing-21d return buckets / fwd 252d`
- **軸名**: 下落の速さ（trailing 1か月リターン）。水準でなく変化率。
- **因果の理由**（検証前に記述）: 速い下落＝過剰恐怖／パニック＝売られ過ぎ→平均回帰。遅いグラインド＝構造的＝正当下落。同じ深さでも速さで挙動が違うはず（憲法の過剰恐怖 vs 正当下落）。
- **対象市場**: S&P500（Yahoo ^GSPC 調整後終値・約30年）

## 結果

- **Part A**（速さバケツ→1年後）：1か月-15%以上の暴落バケツ n=56(延べ) 平均+35.9%・勝率96%・最悪-14.1%（ベースライン+9.4%/78%を圧倒）。遅い下落は逆にベースライン以下。
- **Part B**（深さ5-20%下に固定、速さだけ変える）：速い(1か月-10%超) +18.7%/中央値+28.9% vs 遅い +9.2%/+12.2%。**同じ深さで速い方が約2倍**＝水準でなく速さが信号。
- **Part C**（独立イベント・126日以上離す）：**5イベント**（2001/2002/2008/2011/2020）平均+14.2%・4勝1敗・最悪-14.1%。**leave-one-out +7.5%〜+21.3%＝COVID(+41%)を抜くとベースライン以下。**

## 判定

- **方向性は明確で本物**：速さ＞水準（Part Bが質的証拠）。私の「指数のEVは閉じた」は撤回。
- **だが統計的に未確立**：独立5イベント・COVID依存・4勝1敗は偶然と区別不能。
- 頑健なのは「速い暴落ほど後の最悪が浅い」（worst -14.1% vs -48.8%）＝柱C・リスク整形。
- **最重要の教訓**：1指数・30年では稀少イベントが独立~5個しか取れない（0001も3〜4個）。良い特徴量を増やしてもイベント数は増えない。検出力の壁は**独立イベント数**＝多銘柄/多市場が必須（理由は効率性でなくサンプル数）。

## 地位

**候補（方向性◎・要・多イベント検証）。** 速さ・経路という特徴量は有望。次は同じ特徴量を多数の独立イベント（多銘柄）に当てて、ノイズの床＋OOS＋leave-one-out で確立を試みる。
