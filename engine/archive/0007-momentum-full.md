# 書庫 0007: 全分布モメンタム/反転（直近→将来、非重複・独立）
（信頼設計＝decisions/0006。A の「極端イベントだけ見るな」批判を受けた全分布検証）

- **指紋**: `sp500 monthly / trailing k-mo return -> next k-mo return / non-overlapping`
- **軸名**: 直近リターン→将来リターン（モメンタム/反転）、分布全体
- **因果の理由**（検証前に記述）: 極端イベントは独立~5個で確立不能。分布の本体（普通の動き）は独立サンプルが多く検出力が出る。モメンタム＝群集心理・情報の遅い伝播で強さは続く、反転＝行き過ぎの揺り戻し。どちらが出るか。
- **対象市場**: S&P500（Yahoo ^GSPC・月次361か月）

## 結果（非重複・独立サンプル）

| 時間軸 | n | corr | 解釈 |
|---|---|---|---|
| 1か月→翌1か月 | 359 | +0.003 | 無 |
| 3か月→翌3か月 | 119 | -0.009 | 無 |
| **6か月→翌6か月** | 59 | **+0.245** | **モメンタム（p≈0.06、有意の一歩手前）** |

6か月の単調性：trail下位-12%→fwd-0.6%/win55%、… trail上位+17%→fwd+7.4%/win92%。高い直近→高い将来＝モメンタム。

## 判定

- 短期（1-3か月）はエッジ無し（corr~0）。**6か月モメンタムが唯一の有望リード**。
- 今セッション初めて「偶然と言い切れない」水準（borderline）。**3つの先行結果（200日線0003・velocityの+5%超バケツ・本結果）と整合**＝中期モメンタムは実在の可能性。momentumは学術的にOOS生存の数少ないアノマリー。
- **思想との緊張**：momentum（強さを買う）は柱B＝dip買い・恐怖を買う（平均回帰）と逆向き。短期反転は無く中期モメンタムが生きている、という事実は思想と擦り合わせ要。
- 但し書き：n=59・in-sample・borderline。確立でない。
