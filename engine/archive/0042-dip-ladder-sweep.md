# 書庫 0042: ディップ・ラダー閾値スイープ（-3/5/7/9/15%）の年利
（信頼設計＝decisions/0006。A の問い。0040の閾値スイープ版）

## 結果（S&P500 1996-2026、361ヶ月×10万円。ラダー＝基準を購入価格に更新）

| 閾値 | 発動月 | 平均年利(IRR) |
|---|---|---|
| 脳死・月末 | — | +9.122% |
| -3% | 183/361 | +9.109% |
| -5% | 150/361 | +9.120% |
| -7% | 124/361 | +9.119% |
| -9% | 111/361 | +9.130% |
| -15% | 77/361 | +9.127% |

## 判定

- **全部ほぼ+9.1%（差は年利0.02%以内＝ノイズ）。** どの閾値でも脳死月末と変わらず。
- -3%は51%の月で発動するのにむしろ月末よりわずかに悪い(+9.109%)。深い閾値が気持ち良いが±0.02%＝誤差。
- ＝閾値の問題でなく「月内タイミングという軸そのものにエッジ無し」（0002/0040/0041と一致）。**単一閾値で良し悪しは決まらないが、3〜15%全部並べても同じ＝軸が無効と確定。**
