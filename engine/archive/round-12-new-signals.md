## 個別詳細

### A9: HYG-8% QE以降（2009-03~） ★★★ 採用

```
FULL  n=56  mean=+14.65% DCA差=+12.32% 勝率=95% Z=+12.03
TRAIN n=13  mean=+21.36% DCA差=+19.91% 勝率=100% Z=+8.26
TEST  n=43  mean=+12.62% DCA差=+9.40%  勝率=93% Z=+9.42
```

**採用決定。Round 11 でQE構造変化を定性確認済み。今回の正式Z-score: TEST Z=+9.42。**

- Bonferroni 閾値(3.44) を大幅超過
- TRAIN(100%勝率, n=13) も TEST(93%勝率, n=43) も一貫
- 「QE以降は中央銀行が信用市場を支える → HYGの暴落=買いの確信度上昇」という構造ロジックが裏付けられた
- TRAIN n=13 の少なさは「2009年以降に限定している」ことの必然。期間を絞ったルールで n=13 は許容範囲

**ロジック**: HYG が60日高値から -8% 以下の日 AND ATH-5% 以下 → エントリー（QE開始 2009-03 以降のみ）

### A1b: BB -2.0σ 下抜けクロス（要精査・採用保留）

```
FULL  n=211 mean=+3.21% DCA差=+0.88% 勝率=71% Z=+1.57
TRAIN n=107 mean=+0.62% DCA差=-0.82% 勝率=59% Z=-0.97
TEST  n=104 mean=+5.88% DCA差=+2.66% 勝率=84% Z=+4.10
```

- TEST Z=+4.10 → Bonferroni通過 (3.44超え)
- **しかし TRAIN が逆方向 (DCA差-0.82%, Z=-0.97)** → 過学習または構造変化の可能性
- phi2 や RSI<25 は TRAIN でも一定の正の方向性があった。BB-2.0σ は TRAIN で負
- 判断: **採用保留。Jackknife で TRAIN 弱さの原因を診断してから判断する**

失敗の理由（暫定）: TRAIN期間(1996-2011)でのボリンジャーバンド逸脱が「テクニカル的な底」にならないケースが多い。2000年代は緩やかな上昇相場で、-2.0σ逸脱後に更なる下落継続が頻繁に起きた可能性。

将来の価値: TEST期間(2011-2026)では Z=+4.10 と本物のシグナルである可能性。VIX等の市場状態フィルタと組み合わせれば有効化できるかもしれない。→ 優先度A仮説 N5 として追加

### A2: RSI<25 + ATH-10% 複合（境界）

```
FULL  n=57  mean=+2.97% DCA差=+0.64% 勝率=75% Z=+0.64
TRAIN n=39  mean=+0.64% DCA差=-0.81% 勝率=67% Z=-0.58
TEST  n=18  mean=+8.00% DCA差=+4.78% 勝率=94% Z=+3.17
```

- TEST Z=+3.17 → 閾値(3.44) 未達、不採用
- TEST 勝率94% は魅力的だが n=18 が少なすぎ
- RSI<25 に ATH フィルタを追加すると TEST n が 40→18 に半減
- **ATH フィルタが「RSI<25 シグナルを絞りすぎる」副作用を起こしている**
- RSI<25単独(TEST Z=+3.86)のほうが優秀。ATH条件は邪魔

失敗の理由（タイプD: 冗長・逆効果フィルタ）: RSI<25 は既に大幅な相対的売られすぎを捉えており、ATH 条件を追加しても情報量が増えない。むしろ有効なシグナルを除外してしまう。

将来の価値: 「n=18で勝率94%」という数字は RSI<25+ATH シグナルの純粋な質を示唆している。n が十分に増えれば(50以上) 再検証する価値あり。10年後の再検討候補。

### A3: Δ4 + ATH-10%（不採用）

```
FULL  n=103 mean=+4.26% DCA差=+1.94% 勝率=70% Z=+2.50
TRAIN n=66  mean=+3.20% DCA差=+1.75% 勝率=67% Z=+1.61
TEST  n=37  mean=+6.16% DCA差=+2.94% 勝率=76% Z=+2.75
```

- TEST Z=+2.75 → 閾値未達、不採用
- TRAIN/TEST 両方で正方向は良い性質。ただし確信度不足
- Δ4単独（Round 09）より改善しているが不十分

失敗の理由（タイプC: 有意差不足）: ATH フィルタで方向性は改善されたが、統計的確信度が Bonferroni 閾値に届かなかった。ドットコム・GFC の「連続下落が続く環境」では効果が薄れる可能性。

将来の価値: TRAIN/TEST 両方で DCA差正方向。弱いが存在するシグナル。将来データが増えれば n の自然増加で有意化する可能性。5年後の再検証候補。

### A5: CAPE バリュエーション倍率DCA（不採用）

```
A5 TRAIN: CAPE倍率 vs 均等DCA 差: +0.154%
A5 TEST:  CAPE倍率 vs 均等DCA 差: -0.106%
TEST DCA差: -0.13%, Z=-0.25
```

- CAPE倍率付き DCA と均等 DCA の差が ±0.15% 以内 → 効果なし
- 均等DCA自体の Z がほぼ0（全期間平均をベースラインとしているため当然）

失敗の理由（タイプA: 構造的無効）: 月次 DCA のサイジング調整では 63日後リターンへの影響が薄すぎる。CAPE は「長期10年超」の予測力はあるが「63日後」への効果はほぼない。

将来の価値: CAPE を「63日後への投資判断」ではなく「年次リバランス比率」や「10年累積リターン予測」に使うなら有効かもしれない。計測軸が違う。→ 計測方法変更で再検証候補。

### A10: phi2 AND RSI<25 同時発火（参照シグナル、統計不足）

```

```

- 全期間 n=6、TEST n=2 → 統計的判断不能
- 平均 DCA差+19.52% は phi2単体(+5.99%)の3倍以上
- 「年1回あるかないか」の最高確信度シグナルとして実運用参照は可能
- **採用/不採用の二択でなく「参照シグナル」として扱う**。起きたら×3.0投入を検討
