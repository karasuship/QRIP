## N10: B4 + HYG 非崩落フィルタ — 不採用

```
B4 標準（195件）→ N10（152件）: HYGフィルタで 43件除外
TEST: n=49 DCA差=+4.66% Z=+4.96（閾値通過）
TRAIN: n=103 Z=-0.42（B4と同程度の弱さ）
```

**失敗の理由**: HYGフィルタを追加したにもかかわらず TRAIN Z が -0.70→-0.42 とほぼ変化なし。
GFCのL字型回復時のB4弱点を解消する目的だったが、HYGが崩壊する前の GFC 初期（2007年後半〜2008年前半）の発動は除外できていない。

B4の TRAIN 弱さはHYGで排除できる種類ではなく、phi2本体と同一の構造的問題（L字型回復時の逆効果）を引き継いでいる。N10はB4の劣化版。

## N_LQD: LQD QE後 — 不採用（TEST 未達）

```
LQD-3%: TRAIN n=34 Z=+6.52 / TEST n=199 Z=+2.97（閾値 3.71 未達）
LQD-5%: TRAIN n=16 Z=+7.70 / TEST n=111 Z=-0.13
```

**失敗の理由**: TRAIN の n=34 は QE 以降の全データ（2009-2026）なので信頼性低い（サンプル少）。TEST Z が -3% で 2.97、-5% で-0.13 と全体的に弱い。HYG8 QE後（TEST Z=+9.42）と比較して信号品質が劣る。LQD は HYG より信用ストレスの精度が低い（投資適格 IG は HY より早期警戒性が低い）。
