# ラウンド07: 多重比較補正 -- 選択バイアス検証

## なぜここを調べたか（動機）

80+戦略を試して最良のもの（phi2）を選んだ。この選択バイアスはどれほど深刻か？
Bonferroni補正後でもiota2/phi2は統計有意か？が最後の未解決問題だった。

## 何を試したか

`engine/multi_compare.py`: 段階的発展戦略のMonte Carlo Z スコア比較
- delta cap1 (D2のみ) → iota2 flat → iota2 tiered → phi2 の順に評価
- Bonferroni補正 (80戦略想定)
- 80個の完全ランダム戦略の最大Z分布 (選択バイアス上限)

## 結果

### 段階的発展の Z スコア (TEST 63d, n_sim=2000)

| 戦略 | TRAIN Z | TEST Z | TEST p |
|---|---|---|---|
| delta (D2 cap1, ATH なし) | +0.85 | **+8.06** | 0.0000*** |
| delta (D2 cap6, ATH なし) | +0.83 | **+8.08** | 0.0000*** |
| iota2 flat (ATH-10%+ cap6) | -0.10 | **+9.55** | 0.0000*** |
| iota2 tiered (ティア重み) | +0.05 | **+11.69** | 0.0000*** |
| phi2 (vol 倍率, cap6) | +1.91 | **+26.10** | 0.0000*** |

### Bonferroni 補正

- 80戦略想定: alpha = 0.05/80 = 0.000625 → 閾値 Z > 3.22
- **phi2 の Z=26.10 >> 3.22: 補正後も圧倒的に有意 ***

### 選択バイアス上限試算

80個の完全ランダム戦略を100回試行した最大Z分布:
- 平均 Max Z = 2.37
- 95パーセンタイル = 3.29
- 観測された最大 Max Z = 3.86

**phi2 の Z=26.10 を超えた試行: 0/100 = 0%**

## 何が分かったか

### 発見1: 選択バイアス問題は決定的に解消

80個のランダム戦略でも Max Z は最大3.86。phi2の26.10はその7倍。
ランダムな選択バイアスで達成できるレベルを物理的に超えている。

### 発見2: D2 効果そのものが TEST で強い

一番シンプルな delta (D2 cap1) でも TEST Z=8.06。
これは「ATHフィルタなし、ボーナスなし、ただの急落翌日買い」でも TEST で有意。
→ **2011-2026 の TEST 期間は「急落後リバウンド」という市場構造が特に強かった。**

### 発見3: 段階的設計改善の証拠

Z スコアが delta→iota2→phi2 で単調に増加（8.06→9.55→11.69→26.10）。
各設計決定 (ATH フィルタ追加、ティア重み付け、vol 倍率) が信号を実際に改善した。
「偶然選んだ最良値」なら Z は 1 戦略だけ高く他は低いはず。全部高い。

### 発見4: TRAIN と TEST の分離（正直な評価）

**TRAIN Z (1996-2011)**: 0.85〜1.91（ほぼ有意でない）  
**TEST Z (2011-2026)**: 8.06〜26.10（圧倒的有意）

TRAIN が弱い理由: ドットコム崩壊・金融危機という「L字型回復」のあった弱気相場を含む。
TEST が強い理由: 2011-2026 は歴史的強気相場の一部（COVID を除けば急落は全て短期回復）。

**正直な結論**: ι2/φ2 は TEST（2011-2026 の強気相場環境）で非常に有効で、  
統計的に有意であることは確定的。ただし **TEST 期間が「有利すぎる」環境**だった  
可能性も排除できない。長期弱気相場（日本株1990年代等）では異なる結果になる可能性。

## 研究全体の最終評価

### 確定した主張（高信頼度）
1. D2 効果は TEST で統計的に実在する（Z=8+, p<0.0001）
2. ATH フィルタがシグナル品質を向上させる（メカニズム理解あり）
3. vol>0.25 が追加の情報を持つ（フィールズ理論で支持）
4. 多重比較補正後でも有意（Bonferroni 80テスト補正で Z=26.10 >> 3.22）

### 正直な限界（低信頼度領域）
1. TRAIN（2001-2011）では有意でない → 長期弱気相場では機能が弱まる
2. TEST 期間の特殊性（歴史的強気相場）が結果を過大評価している可能性
3. 日本市場・欧州市場には別途検証が必要
