## 発見と解釈

### 発見1: RSI<25 の TRAIN 弱さは「GFC」ではなく「ドットコム」が主因

**仮説は覆った。**

- GFC を除外しても DCA差は +0.48% → +0.40%（変化ほぼなし）
- ドットコムを除外すると DCA差は +0.48% → **+2.19%**（大幅改善）

つまり RSI<25 の TRAIN 弱さの主因はドットコム期間（2000〜2002）だった。

**なぜドットコムが弱いのか**：
ドットコム崩壊期の RSI<25 トリガーは n=20 で DCA差=+1.01%。DCA差は正なので「RSI<25 が DCA に負けている」わけではない。問題は「ドットコム期間の DCA差（+1.01%）が非危機期間の DCA差（+3.10%）より低い」こと。
つまり「RSI<25 がドットコムで失敗した」のではなく、「ドットコム期間は非危機期より相対的に弱い」というだけで、全期間を通して RSI<25 は DCA を上回り続けている。

### 発見2: GFC では RSI<25 が DCA より「マシ」（予想外）

GFC 期間のみ: n=7、mean=-4.07%、DCA=-7.29%、**差=+3.22%**

6/7 回のトリガーが 63d 後もマイナス（勝率14%）。しかし「損をしたがDCAよりは損が少なかった」。GFC の L 字型回復では「どこで買っても 63d 後は損」だが、RSI<25 でのタイミングは DCA より 3.22% 少ない損傷で済んだ。

**iota2 との比較**：
- iota2 の GFC：差 -0.27%（DCA よりわずかに悪い）
- RSI<25 の GFC：差 +3.22%（DCA より良い！）

RSI<25 は GFC 期間でも DCA に「勝って」いた。これは iota2 より GFC 耐性が高い可能性を示す。

### 発見3: TEST は COVID/2022 に非依存（頑健性確認）

COVID と 2022 を両方除外しても DCA差は +3.92% → +4.21%（さらに改善）。
TEST の優位は特定イベントへの依存ではなく、2011-2026 全般の V 字型回復環境が貢献。

### 発見4: HYG-8% は QE で構造が反転した（仮説確定）

| | QE 以前 | QE 以降 |
|---|---|---|
| DCA差 | **-7.72%** | **+11.21%** |
| 勝率 | 21% | 95% |

**差が 18.93%pt。** これは完全な構造変化を示している。

QE 以前（〜2009-03）: HYG-8% 下落 = 「信用市場の根本崩壊・企業倒産連鎖」= 株も長期低迷する本物の危機。n=42 のほとんどが 2007-2009 GFC 期間に集中。
QE 以降（2009-03〜）: HYG-8% 下落 = 「信用市場の一時的ストレス」= FRB が必ず介入・緩和する。結果として「信用が売られた = 株の絶好の買い場」という構造になった。

**「HYG-8% は QE 以降専用ルール」として再定義可能。** ただし「次の QE 転換点」を事前に知ることはできないため、QE 依存ルールとして慎重に扱う必要がある。

## 戦略への含意

### RSI<25 の採用妥当性（再確認）

| 期間 | 差（RSI<25 - DCA） | 備考 |
|---|---|---|
| ドットコムのみ | +1.01% | 正（ただし非危機より低い） |
| GFC のみ | +3.22% | 正（DCA より損が少ない） |
| 非危機 TRAIN | +3.10% | 強い |
| TEST 全体 | +3.92% | 強い |
| TEST (COVID/2022 除外) | +4.21% | さらに強い |

**全サブ期間で RSI<25 は DCA を上回っている。** TRAIN Z=0.39 が「ランダム相当」に見えた理由は、サブ期間の DCA差のばらつき（+1.01% から +3.22%）で Z 値が稀釈されたため。**構造的欠陥ではない。採用判断を維持。**

### HYG-8% の条件付き採用

「QE 継続環境（中央銀行が積極介入するレジーム）」という前提付きで再定義すれば、HYG-8% は極めて強いシグナル（QE 以降 DCA差 +11.21%、勝率95%）。ただしレジーム変化（QE 終了・FRB 緊縮転換）で破綻するリスクがある。**条件付き採用候補として 01-二次的な核に記録する価値あり。**
