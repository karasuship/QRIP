# 書庫 0030: 生存者バイアス感度分析 ＝ 深い下落買いのエッジは幻
（信頼設計＝decisions/0006。0029のpromisingな結果に、データ無しで survivorship の壁を測る）

## 何をしたか

押し目買い(-30%下落→6ヶ月)のエッジを全銘柄でプール集計し、「廃止株（-100%）を割合p追加したらエッジがいつ消えるか（損益分岐の追加廃止率）」を算出。

## 結果

- プール：押し目fwd **+13.7%** / baseline **+13.2%** / **エッジ +0.5%**（per-stock等加重平均+3.5%は過大評価。実体は+0.5%）。
- **損益分岐の追加廃止率 ≈ 1%**。
  - +1% → -0.6%（消滅）／+3% → -2.8%／+5% → -4.9%。

## 判定（最も深い正直な結論）

1. per-stock+3.5%は等加重の過大評価。プール（全押し目を買う）実体は**+0.5%のみ**。
2. その+0.5%は**わずか1%の追加廃止率で消える**。
3. -30%窮地の銘柄は平均より遥かに高率で上場廃止（数%〜10%超/年）＝1%を軽く超える。
4. **＝深い下落買いのエッジは、ほぼ確実に生存者バイアスの幻**（生き残り銘柄が構造上必ず回復してただけ）。

## EV研究の到達点

- 憲法の核心「ボラ銘柄の恐怖を買う」は、生存者補正で消える。個別銘柄まで降り、アマチュアが飛ばす生存者補正をかけ、エッジは蒸発。
- 唯一頑健なr5反転もコストで消える。
- **＝生存者＋コストを両方生き残る、個人が取引できる生EVエッジは無い**（効率の低い隅まで探しても）。

## 意味（失敗でなく最深の確認）
30検証＋個別銘柄＋生存者補正の徹底が、**憲法の核心「予測に賭けるな・価値は柱A/C」を最も深いレベルで実証**。"出し抜かれない側"（柱C＝出荷済み製品）が正しい賭けだと実験で証明しきった。EV研究はここで rigorous な terminus（無料データ＋生存者補正の範囲で）。さらに進むには有料の上場廃止込みデータ＋それでも薄い・コスト感応＝学生2人の労力を賭ける価値は低い。
