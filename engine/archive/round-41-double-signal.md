## 結論

### F1: 二重シグナルは独立戦略として使えない（n 不足）、用途は「特注意フラグ」

**重要発見 1: RSI<25 単体は統計的にほぼ無意味（TRAIN Z=+0.06）**

RSI<25 only（phi2 条件を満たさないケース）は TRAIN でランダムと区別がつかない。
「RSI が 25 を下抜けた」だけでは信頼できる買いシグナルにならない。

RSI<25 が OR 体制で機能しているのは、phi2 v3 の ATH-10%/vol/CRS 条件と
**実質的に近い局面（大型下落 + 恐怖指数高）で同時に発生するケース**に絞られる。

**重要発見 2: phi2 v3 only が最高品質（TEST Z=+7.39、+13.6%）**

phi2 v3 のみ（RSI が 25 未満でない）のグループが最も安定して高い Z スコアを示す。
RSI まで 25 を割るような極端なパニック局面は、phi2 単体より必ずしも優れない。

**重要発見 3: 二重シグナルはコロナ・震災級パニックの指標**

TEST の 3 件: 2020-03（コロナ暴落 ×2）+ 2025-04
TRAIN の 5 件: 1997-10（アジア通貨危機）/ 2000-04（ITバブル）/ 2008-10・2009-02（GFC）/ 2011-08

全て「後から振り返ると絶好の買い場」。ただし n=8 では統計的有意性を確認できない。

**採用方針: サービス上の「特注意」フラグとして使う**

- phi2 v3 + RSI<25 同時 → 「超高品質アラート（過去 30 年 8 回）」として表示
- 独立した取引ルールとしては n が不十分で使えない
- CRS=5 + phi2 + RSI<25 の三重条件はさらに稀（検証不要なレベル）

### OR 体制のロジック整理

```
phi2 v3 OR RSI<25: TEST Z=+7.66

内訳:
  phi2 v3 only (n=20): Z=+7.39  -- 高品質 backbone
  RSI<25 only (n=35):  Z=+3.06  -- 補完的。phi2 ほどは強くない
  phi2 AND RSI<25 (n=3): Z=skip  -- 最高品質だが n 不足
```

OR 体制の Z は phi2 単体と似た水準（+7.66 vs +7.39）であり、
RSI<25 alone の追加が Z を「稀薄化」している面もある。
RSI<25 の貢献は「テールリスクカバー（phi2 が見逃す局面を補う）」であり、alpha 増幅ではない。
