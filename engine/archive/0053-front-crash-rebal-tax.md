# 書庫 0053: 暴落へ前借り（総額固定）＆ リバランスの税・手数料ドラッグ
（信頼設計＝decisions/0006。A「暴落で前倒し投資→後で休む方式が良さそう」「リバランス無は手数料無で良いと聞いた」）

## ① 暴落へ前借り→後で休む（総額同じ・0%借入＝最良ケース）

| 戦略 | 平均年利(IRR) | 脳死比 |
|---|---|---|
| 脳死フラット | +9.122% | — |
| 暴落-10%で6ヶ月分前借り(25回) | +9.090% | -0.033% |
| 暴落-10%で12ヶ月分前借り(16回) | +9.137% | +0.014% |
| 暴落-20%で6ヶ月分前借り(15回) | +9.097% | -0.026% |
| 暴落-20%で12ヶ月分前借り(10回) | +9.019% | -0.103% |
| [対照]暦で40ヶ月毎に12分前借り | +9.217% | +0.095% |

- A案＝ノイズ、むしろマイナス寄り。**暴落を狙わない暦ベースの対照の方が高い**＝微差の正体は「少し早く市場に入れた(time-in-market)」だけで暴落タイミングは無価値。-20%で弾を撃ち尽くしその後の底を取り逃す＝逆効果（2008型）。
- この僅差すら0%借入(タダのレバ)前提。現実の信用金利(〜7%)で全て深いマイナス。

## ② リバランスの税・手数料（36銘柄・生存バイアス有）

| 戦略 | 平均年利(IRR) |
|---|---|
| リバランスなし | +21.364% |
| 年1リバランス・コスト0 | +25.370% |
| 年1リバランス・税20%+手数料0.1% | +22.695% |

- Aの直感は正しい：税・手数料が年-2.7%食う（売却益課税は巨大）。
- だし税引後でもリバランス有(+22.70%)>無(+21.36%)＝「手数料無だから無が勝つ」は本サンプルでは不成立。かつ両者とも生存バイアスの幻。
- **現実解＝広い指数ファンドを保有し個人はリバランスしない**：ファンド内部で無税・極低コストの自動再構成。「集中 or 税ドラッグ」を両取り。

## 結論
- **暴落への前借り(固定予算)は脳死を超えない**（0%借入でもノイズ〜負、暦対照に劣る）。利は時間効果のみで、暴落タイミングは無価値かつ金利で消える。
- **リバランスの税・手数料は実在(年-2.7%)＝Aの直感は正**。だが答えは「個別を組むかどうか」でなく**広い指数ファンドを持ち自分では触らない**こと（無税・自動入替）。
- ＝中心命題を再々確認：固定予算の組み替えでは脳死DCAを超えられない／勝ち筋は time-in-market・低コスト・低税・暴落を耐える行動（柱C）。
