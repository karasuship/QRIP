# 書庫 0052: 個別銘柄リバランス有無 ＆ 暴落比重DCA（固定予算）
（信頼設計＝decisions/0006。A「銘柄持ってリバランスしない方が利回り良いか」「暴落に比重置けば脳死より高いのでは」）

## Q1: バスケット リバランス有無（36銘柄・生存バイアス有）

| 戦略 | 平均年利(IRR) |
|---|---|
| 指数(S&P500)にDCA | +9.122% |
| バスケット・リバランスなし | +21.364% |
| バスケット・年1リバランス | +25.370% |

- 「リバランスしない方が良い」は**不成立**＝むしろリバランス有が+4.0%/年高い（下がった銘柄への買い直し＝内部押し目買いが回復で報われた）。
- だが+21/+25%は**全て生存バイアスの幻**。36銘柄は30年生き残った勝ち組の後出し選択。実S&P500（退場込み）は+9.1%。リバランス勝ちも「下落銘柄が必ず生存」と知ってる前提＝現実では倒産株への買い増し＝破滅。
- **事前に勝者は選べない以上、両者とも再現不能・水準は無意味。**

## Q2: 暴落に比重を置く（固定予算）vs 脳死フラット

| 戦略 | 平均年利(IRR) |
|---|---|
| 脳死フラット(10万/月) | +9.122% |
| 暴落比重 w=1+|下落|（神の目=look-ahead） | +9.206% |
| 暴落比重 w=1+2|下落|（神の目） | +9.271% |
| 暴落比重 w=1+4|下落|（神の目） | +9.365% |
| 現金温存→-20%で全投下（現実的） | +8.831%（脳死に負け） |

- 未来の全暴落を知る神の目で比重最大化しても+0.08〜0.24%＝ノイズ（しかも上限／カンニング込み）。
- 現実的な現金温存→暴落投下は-0.29%で負け（待機現金＝機会損失）。

## 結論
- **リバランス有無の問いは生存バイアス無しデータ無しには答えられず、水準は幻**。勝者を事前に選べないのが本質。
- **固定予算で暴落に比重を置いても脳死を超えない**（神の目でも+0.2%）。超えるのは暴落時に追加資本/レバを入れる時だけ＝書庫0050/0051のレバ・レジーム賭け（比重の妙でなくリスク増幅）。
- ＝Q1・Q2とも、中心命題「固定予算の組み替えでは脳死DCAを超えられない／高リターンはリスク×行動の対価」を再確認。
