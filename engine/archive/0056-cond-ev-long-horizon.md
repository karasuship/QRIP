# 書庫 0056: Y変更（63d→252d）— CAPEは遅いシグナルか？

（検証軸②。0054のCAPE条件を時間軸を延ばして再検証。仮説：CAPEは1年以上で効く）

- **対象**: S&P500 × Y=63/126/252d × X=5/10/15/20%
- **条件**: UNC(無条件) / C_HI(CAPE>p75) / C_LO(CAPE<p25) / C_HI+VL(CAPE>p75×rvol_low)
- **OOS分割**: TRAIN=1996-2011 / TEST=2011-2026

## 事前登録した仮説

CAPE高 = 将来期待リターン低 = Y=252dのドリフトが無条件より低い  
→ 同じP(drop)でもEVが改善する可能性（特にY=252dで効果が現れるはず）

## 判定

**仮説は部分的に成立した。ただし実用水準に届かない。**

1. CAPEは確かにY=252dで「ドリフトの低下」を示す:
   - C_HI X=20% TEST: drift=+10.5% vs UNC +14.4%（-3.9pp）
   - C_HI+VL X=20% TEST: drift=+6.9% vs UNC +14.4%（-7.5pp）

2. ただしP(drop)も上昇する（連動は続く）:
   - C_HI X=5% TEST: P=69% vs UNC 49%
   - 下落確率が上がる = EV改善と相殺

3. EV正転ゼロ。最良値:
   - C_HI+VL Y=252d X=20% TEST: EV=-8.16%（vs UNC=-15.38%、+7.2pp改善）
   - n=19（OOS）で統計的に薄い

4. Shillerのが予測する「CAPE高=将来リターン低」は弱く確認された（drift低下）が、  
   同時にP(drop)上昇が相殺する。時間軸を延ばしても構造的連動は切れない。

5. C_LO（CAPE低、バーゲン相場）はEVが逆に悪化:  
   低P(drop)だがdriftも高い → 待機コストが大きい → 最悪の選択肢

## 次軸

③ドリフト予測の直接アプローチ:
  - これまでの全検証でEVの足を引っ張っているのは「drift（機会損失）」の大きさ
  - 直接「次のY日間のdriftが小さい期間」を予測できれば、待機戦略が成立する余地がある
  - CAPEはdriftの低下と相関があると確認された → drift回帰モデルへ応用可能
