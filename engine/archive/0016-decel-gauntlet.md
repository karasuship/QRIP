# 書庫 0016: 減速シグナルの関門 + 速さ×ボラの組み合わせ×ラグ
（信頼設計＝decisions/0006。A「減速を厳密検証＋組み合わせもラグで」）

- **対象**: S&P500（Yahoo ^GSPC 日次30年）

## PART A. 減速(decel)シグナルの関門

シグナル＝5日急落(≤-3%)→平坦化(≥-0.5%)。forward 45d。n=408（豊富）。

| | 値 |
|---|---|
| decel 平均 | +2.27% |
| ベースライン | +1.69% |
| 超過 | +0.58% |
| ノイズの床 平均/p90/p95 | +1.68% / +2.11% / +2.21% |
| **床判定** | **p95超え＝EDGE** |
| **OOS 前半/後半** | **+0.46%（ベースライン以下！） / +4.09%** |
| コスト0.2%引き後 超過 | +0.38% |

- **ノイズの床はp95で超えた（ランダムより有意）が、OOSで死亡**：前半はベースライン以下、エッジは後半だけ。
- **教訓**：「ノイズの床を超える」でも足りない。**OOSがより厳しい関門**。in-sample有意・OOSで消える（0008モメンタムと同型）。指数で唯一データの足りてた"変化率"候補も**未確立**。

## PART B. 速い暴落のラグ × ボラレジーム（組み合わせ）

| regime | n | 1d→45d→60d 超過 |
|---|---|---|
| high-vol | 161 | +0.4 → +5.9 → +5.7 |
| low/mid | **1** | （サンプル無し） |

- **速い暴落（-10%/月）＝ほぼ必ず高ボラ事象＝両者は共線で分離不能**（off-cell n=1）。
- 45日ラグは"1つの現象"。速さ×ボラの組み合わせは情報を足さない。＝組み合わせの限界の正直な発見。

## 判定

- decel：**未確立（OOS失敗）**。床は超えたがOOSで死。
- 速さ×ボラ：**共線で分離不能**。
- ＝指数で well-powered な最後の候補（decel）もOOSで落ちた。指数EVの決着（0014）をさらに補強。

## 地位

**減速＝不採用（OOS失敗）。** 指数EVは well-powered ケースでも閉。45日ラグ（0015）は柱C素材として残る。EV確立は多銘柄のみ。
