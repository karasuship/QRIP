# 書庫 0012: レジーム条件づけ探索（実現ボラ × 反転/モメンタム）
（信頼設計＝decisions/0006。非定常性の規律つき探索。意味は後付け・OOS厳格・全試行記録＝多重検定勘定）

- **指紋**: `sp500_daily / realized-vol-tercile regime x trailing-k vs next-k return / k in {1,5,21}`
- **軸名**: レジーム（実現ボラ高/中/低）× 短期反転/モメンタム
- **規律**: レジーム＝リアルタイム観測（trailing21d実現ボラ）の事前変数。各レジームを EARLY/LATE 時間分割＝別々の出現でOOS。SURVIVOR＝両半期で同符号かつ|corr|>0.10。意味は survivor にのみ後付け。
- **対象**: S&P500（Yahoo ^GSPC・日次30年）

## 判定

- **確立はしていない**（厳密掟で生存ゼロ）。
- **だが方向性は本物っぽい**：「**高ボラ時に反転が強まる**」が全3時間軸・両半期とも負＝符号一貫。low/midは弱い。偶然の単発でなくコヒーレント。事前仮説(α)「パニック→行き過ぎ→跳ね返り」と一致。
- **＝局所法則（レジーム偏在）の最初の支持証拠**（A の非定常性論を規律内で支持）。
- 但し：magnitude控えめ／後半で減衰（high vol 21d late -0.046）／1〜5日反転はコストで消える。**コストに耐える候補は「高ボラ×21日反転」のみ**。

## 地位

**弱い候補（高ボラ×21日反転）／他は不採用。** 次：高ボラ×21日反転を、(1)個別の高ボラ出現（1998/2002/2008/2011/2020/2022）で各々成立するか (2)コスト引き後にノイズ床を超えるか (3)VIX取得で regime を照合。
