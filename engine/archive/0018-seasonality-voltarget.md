# 書庫 0018: 季節性 + ボラ・ターゲティング（well-powered な正の候補）
（信頼設計＝decisions/0006。A「プラスを見つけたい・時間軸/組み合わせ/法則はまだある」）

- **対象**: S&P500（Yahoo ^GSPC 日次30年）

## 1. 季節性「ベスト6か月」（Nov-Apr 投資 / May-Oct 現金）

| | 倍率 | CAGR | maxDD | 在市 |
|---|---|---|---|---|
| buy&hold | 11.3x | +8.4% | -57% | 100% |
| 季節性 | 5.6x | +5.9% | -38% | 49% |
| 季節性 net 0.1%/switch | 5.3x | +5.7% | | |

OOS：前半 b&h+4.5%/季節+5.0%（勝）、**後半 b&h+12.3%/季節+6.8%（大敗）**。
→ **OOS失敗**（夏に暴落が多い時代のみ有効）。リスクは減るが生リターンは負け。正のEVでない。（現金月のT-bill利息は未加算＝加えれば差は縮む）

## 2. ボラ・ターゲティング（月次露出=min(1, median_vol/trailing_vol)）

| | CAGR | vol | maxDD | ret/vol(crude Sharpe) |
|---|---|---|---|---|
| buy&hold | +8.6% | 19.3% | -57% | 0.44 |
| vol-target | +7.1% | 13.9% | -46% | **0.51** |

→ 生リターンは低い(+7.1<+8.6)が、**シャープ 0.44→0.51（+16%）改善**。ボラ・クラスタリング（ボラは予測可能＝最も頑健な事実）に基づくのでOOS安定の見込み。**今セッション初の頑健な"正"——ただし"EVの正"でなく"シャープの正"。**

## 判定・価値

- 季節性：**不採用（OOS失敗・リスク寄り）**。
- ボラ・ターゲティング：**シャープの正（採用候補）**。2つの価値：
  1. **製品直結（柱A/C）**：露出を下げて DD半減・滑らかな乗り心地＝**狼狽売りが減る**。
  2. **生EVへの唯一の橋**：高シャープをレバレッジで buy&hold のボラまで上げれば同リスク高リターンになりうる。ただしレバはコスト・リスク・「狼狽者向け」と逆行＝要慎重・未検証。

## 地位

**ボラ・ターゲティング＝シャープ改善で採用候補（柱A/C＝乗り心地）。生EVの正は引き続き未発見だが、レバ版が唯一の未検証の橋。** 季節性＝不採用。
