# 書庫 0024: 上側パターン（天井・買われすぎ・高値・連騰）＝上下の非対称
（信頼設計＝decisions/0006。A「特徴的な上がり方・天井・何%上がると天井か・規則性を探したのか」）

- **対象**: S&P500（Yahoo ^GSPC 日次30年）。baseline fwd21d=+0.79%/win64%。

## 結果

| 検証 | 結果 |
|---|---|
| A 直近21d上昇幅→21d後 | +10〜20%上昇後 +1.37%(win71%)＝継続(モメンタム)。天井なし。<-10%下落だけ+4.44%(反発) |
| B MA200から+10〜20%上 | +0.43%(win62%)＝わずかに伸び悩むが弱い |
| C 1年高値更新→21d後 | **+0.10%(win60%)** vs baseline+0.79%＝唯一「高値で失速」の弱い兆し |
| D 4連騰後 翌日 | -0.09%＝極小・コストで消える |

## 判定（重要な構造）

- **上側に明確な天井は無い＝上側は効率的**。「何%上がると天井か」→系統的な天井は出ない（むしろ継続）。
- **＝上下の非対称が確定**：下側＝恐怖の過剰反応（売られすぎ→反発、活用余地）／上側＝効率的（greedは行き過ぎを作らない）。
- 機会は**恐怖の側のみ**。憲法「恐怖を買う」が上側検証でも裏付いた。
- 唯一の弱い兆し＝1年高値更新後の失速（要OOS）。

## 地位
天井/買われすぎ/連騰＝**不採用（上側は効率的）**。非対称（機会は恐怖側）＝構造として確定、柱B/Cを研ぐ。1年高値の失速＝弱い要OOS候補。
