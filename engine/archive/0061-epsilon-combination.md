## なぜD1は劣るか

- D1（-1%閾値）: TEST bonus_ROI=2.8989 = DCAより3%低い
- 1%の日次下落は月平均0.74回発生（ノイズが多い）
- 急落の「意味がある」局面を弁別できていない
- D2（-2%）以上に絞ることでシグナルの質が上がる

## δ vs WAIT戦略の根本的な違い

| | WAIT戦略（0052〜0059） | δ戦略（D2） |
|---|---|---|
| タイミング | 月初から「待つ」 | 急落が「来た後」買う |
| 待機コスト | 常に発生（EV負） | ゼロ（すでに落ちた） |
| DCA対比 | ≈同等（0059） | **+1.75%/ボーナス円（OOS）** |
| 直感 | 安値を「予測」 | 安値を「後追い確認」 |

## 複合の結論

「WAIT_W + D2」の組合せは効果なし（W条件月 = 低ボラ = 急落が来ない → 両者が反相関）

独立に積み重ねられる組合せが見つかっていない。現状の最良複合:
→ 毎月DCA + -2%急落日に0.5単位追加 = **TEST +0.25%、FULL +1.46%**

## 実装判断

D2戦略（-2%急落買い増し）は暫定採用を検討できる水準:
- OOS一貫性: ○
- 実装容易性: ○（前日比-2%を毎日チェックして月1回追加投入）
- 必要追加資本: 月次積立×33%程度の予備（TEST 210/180）
- 懸念: 月次ベースの制約のため大クラッシュ時に1回しか使えない
