# Round 35: A5v2 CAPE 長期軸リバランス（2026-06-22）

## 仮説

CAPE < 20 の時に phi2 が発動すると、63日後・1年後リターンが特に高い。
「割安市場での暴落買い」は「割高市場での暴落買い」より優れるか？

---

## 結論

### A5v2 不採用（3つの理由）

**理由1: TEST 期間に CAPE < 20 のデータが存在しない**  
2010年以降、市場の CAPE は 20 を下回っていない。仮説を TEST で検証できない。

**理由2: CAPE ストラテジフィケーションは age/CRS と相関**  
- CAPE 20-25（TRAIN Z=-2.31）= GFC後の L字期間（age 91-252）と重複
- age フィルタ済み phi2 v3 を CAPE 別に見ると差は縮小する
- CAPE は「何年のデータか」を示すにすぎず、phi2 の品質を独立に示さない

**理由3: 高 CAPE 環境が TEST での phi2 最強ゾーン（逆説）**  
- CAPE 30-35: TEST Z=+6.72（63日）、TEST Z=+9.62（252日）
- QE 以降の「高 CAPE 市場」では、暴落後の回復が強い
- CAPE 高い = 危険 という直感は phi2 タイミングには当てはまらない

### 重要な副次発見

CAPE は長期的な市場の割高・割安を示す指標として有効だが、
phi2（暴落買い）の品質フィルタとしては機能しない。
phi2 の品質は CAPE より VIX/HYG/age（恐怖の深さ）の方が説明力が高い（CRS として既に採用済み）。
