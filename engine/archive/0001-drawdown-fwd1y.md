# 書庫 0001: 下落幅 → 1年後リターン分布
（信頼設計＝decisions/0006。素性ラベル付き。成功も失敗も残す＝地雷②）

- **指紋**: `sp500_adjclose / drawdown<=-X / hold=252d / fwd_return`
- **軸名**: チャート（ピークからの下落幅）
- **パラメータ**: X ∈ {10%, 20%, 30%}、保有 H=252取引日（約1年）
- **因果の理由**（検証前に記述・地雷①）: 深い下落＝恐怖/過剰反応で、その真の持続影響より売られ過ぎ → 平均回帰。深いほどエッジが大きいはず（柱B / Step0）。
- **対象市場**: S&P500（Yahoo ^GSPC 調整後終値・配当込みに近い）
- **期間**: 1996-06-19 → 2026-06-18（約30年）。OOS＝行ベースで前半/後半に分割。
- **試行回数の文脈**: 事前登録した3閾値（有意性狙いのスイープではない）。全結果を記載。

## OOS判定・解釈

- **方向性は仮説と一致**：深い下落ほど 平均↑・勝率↑・最悪が浅くなる。
- **だが統計的に未確立**：
  - -10%は43%の日が該当＝signalでなく日常。ベースラインと差なし。
  - -30%の延べ763日は**独立イベント2002/2008/2020の3〜4回**に過ぎない（重複ウィンドウ水増し）。
  - OOSのtest側がほぼ空（-30%でn=2）＝**OOS不成立**。後半15年は深い下落が稀。
  - 30年上昇相場では「-30%から1年保有で回復」はほぼ同義反復になりうる。

## 地位

**候補（弱い）** — 因果の筋と方向性はあるが、独立サンプル不足・OOS不成立。「生存」には昇格しない。
**次の関門**: 多市場（TOPIX・各国・セクター）で独立イベントを増やす／重複を畳んでイベント単位で数える／OOSを地合い・イベント単位に。最も頑健なのは「深い下落ほど最悪が浅くなる」（柱C・予測非依存）。
