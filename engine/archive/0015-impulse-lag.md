# 書庫 0015: インパルス応答＝信号が織り込まれるまでのラグ構造
（信頼設計＝decisions/0006。A「変化率・トレンドの市場合理的反映までのラグは未検証／今のは既知帰結の検算にすぎない」を受けた、ラグ構造そのものの検証）

- **指紋**: `sp500_daily / signal -> cumulative forward excess return path vs baseline / horizons 1..60d`
- **狙い**: 信号後に価格が"正しい水準"へ寄る軌跡を日単位で追い、織り込み完了までのラグを出す。即座=効率的／数週ドリフト=under-reactionラグ。

## 判定（A のラグ仮説への直接の答え）

1. **トレンド転換＝ドリフト無し＝即座に織り込み（効率的）**。"トレンド"側は支持されず（null）。
2. **変化率=減速＝小さいラグ**：最初5日は微下げ→30-60日で+0.7%へ緩くdrift。Aの「数日ずらす」の核はここ（減速直後は沈む）。
3. **速い暴落＝織り込みに約45営業日（≈2か月）**。超過がday10→45で+2.0→+5.9と着実に積み上がり頭打ち。**＝"数日ずらす"でなく"再評価が約2か月かけて完了する窓"が真の構造**。粗い遅延テスト(0-21d)はラグ本体(10-45d)を見逃していた。

## 価値

- **新しい具体的構造**（既知帰結の検算でなく）：信号別の織り込みラグ＝trend即時／decel緩い／fastdrop~45日。
- **柱C素材**：「速い暴落後の反発は数週かけて来た（~45日）。最初の数週で投げるな」＝狼狽売り防止の時間軸の根拠。
- 但し：fastdrop独立~10・overlap＝magnitude未確立（コスト検証0014でノイズ内も既知）。確立でなく"ラグの形"の発見。
