# Round 28: P9 Crisis Recovery Score（CRS）（2026-06-22）

## 仮説

R21-R25 で確立した「恐怖継続中パターン」を数値化する。
複数指標の複合スコア（CRS）でphi2 発動の品質を推定し、
スコアに応じて買い強度を変えることで期待値を改善できる。

---

## CRS の設計

各成分は0/1の二値。合計 0〜6 点。

| 成分 | 条件 | 意味 |
|---|---|---|
| C1 | VIX > 30 | 絶対的パニック水準 |
| C2 | HYG 3日リターン <= 0 | ジャンク債継続下落（危機継続） |
| C3 | DXY 5日上昇 | ドル高継続（流動性逼迫） |
| C4 | age_ath <= 90 | ATH から90日以内（新鮮パニック） |
| C5 | HYG 60日高値から-8%以上 | HYG崩落フラグ |
| C6 | RSP 5日相対リターン <= 0 | 等加重弱（市場幅の悪化） |

### CRS >= 4 のイベント一覧（主要点）

TRAIN（25件）: 大半が GFC 期間

- この二分が TRAIN Z をゼロ近辺に押し込んでいる

- 2022年: 混在（-4% to +6%）

## 結論

### CRS は「買ってはいけない局面」を特定するフィルタとして有効

| 用途 | 推奨 |
|---|---|
| CRS <= 1 はスキップ | TRAIN Z=-2.85。phi2 でも買わない |
| CRS >= 2 は標準 | TRAIN Z=+1.86（phi2 全体より改善） |
| CRS >= 4 は積極買い | TEST Z=+6.32、TEST 勝率88% |

### CRS の TRAIN Z が低い根本理由

GFC 期間（2008-12 まで）の CRS >= 4 トリガーが大量に存在し、全て負ける。
これは「GFC 期間中は恐怖が継続し CRS が高くなる」のに「63日ホライズンでは底が見えない」という構造的問題。
R20（多重ホライズン）で示した通り、126日後なら GFC期間でも phi2 はプラスになる。
つまり **CRS の指摘は正しい。ただしホライズンが短すぎる（63日）だけ**。

### 実装推奨: CRS スコアを phi2 の buy multiplier に連動

```
CRS=0:     phi2 スキップ（または ×0.25）
CRS=1:     ×0.5（慎重）
CRS=2〜3: ×1.0（標準）
CRS=4〜5: ×1.5〜×2.0（積極）
CRS=6:     ×2.5（最大）
```

### CRS=6 は最高品質（TEST n=3、全て+19%以上）

3件全て 2020年3月 COVID 急落（+19%, +28%, +21%）。
VIX>50, HYG崩落, DXY急騰, age<10, 全条件一致。
「歴史的危機の直撃時」が最高品質。
