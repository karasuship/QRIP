## 結論

### phi2 v3 の仮定義（採用候補）

```
phi2 v3 = phi2 v2 AND age_ath NOT IN [91, 252] AND CRS >= 2
```

- **TRAIN Z=+4.55**（Bonferroni 閾値 Z~3.80 を超えた！）
- TEST Z=+7.98（phi2 v2 の +8.04 とほぼ同等 → TEST での後退なし）
- GFC除外 TRAIN Z=+5.05（GFC が消えても強い）

### 2つのフィルタの効果の内訳

| フィルタ | TRAIN Z改善 |
|---|---|
| age スキップのみ | -0.16 → +3.30（+3.46） |
| CRS>=2 のみ | -0.16 → +1.89（+2.05） |
| 両方 | -0.16 → **+4.55（+4.71）** |

相乗効果がある。age スキップは「Lゾーンを除外」、CRS>=2 は「恐怖回復兆候を除外」。
この2つは別の危険を除いているため独立した効果を持つ。

### Bonferroni の観点

累計~290テスト → 閾値 Z~3.80。phi2 v3 TRAIN Z=+4.55 は超えている。
ただし: この仮説は R26（age）+ R28（CRS）の副産物の組み合わせ。
「フィルタを足していった結果」としての側面があるため、真の独立テストとは言えない。
R30 で4体制への適用を確認し、そこで最終判断する。
